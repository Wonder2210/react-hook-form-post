## Getting Started

This is the code used on the react hook form repo , feel free to clone it , to guide yourself through react hook form

### Prerequisites

* Yarn
* Node (v14+)

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Wonder2210/react-hook-form-post
   ```
2. Install NPM packages
   ```sh
   yarn install
   ```
3. Start the demo
   ```sh
   yarn start
   ```
